======================
Unix-specific Services
======================

Although the Python interpreter is very portable, and many of the
standard library modules support multiple platforms, there are some
modules that provide access to platform-specific features. The modules
described in this section access features only available on Unix-based
systems such as Linux, macOS, FreeBSD, and OpenBSD.
